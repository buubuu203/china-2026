## Deploy lên Vercel
1. Push repo lên GitHub
2. Vào [vercel.com](https://vercel.com) → **Add New Project** → Import repo
3. Vercel tự detect static site → click **Deploy**
4. Xong — nhận link `*.vercel.app`

## Cập nhật plan
Chỉnh file `index.html` → commit → push → Vercel tự redeploy.
